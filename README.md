# Telecom Customer & Network Performance Analysis

# Project Overview
This project analyzes telecom customer data, network usage, complaints, and billing behavior to understand network performance and customer risk patterns.
Using tools like Microsoft Excel, SQL, Python, and Tableau, the project focuses on identifying network issues, analyzing customer usage patterns, and generating business insights for telecom service improvement.

# Dataset Overview
The project uses four datasets:
* **customers** – Customer demographic information and subscription type
* **network_usage** – Daily data usage and call activity of customers
* **network_issues** – Network complaints and issue resolution time
* **billing** – Monthly billing details and payment behavior

# Tools & Technologies
* Microsoft Excel
* SQL
* Python
* Pandas
* Matplotlib
* Seaborn
* Tableau

# Data Preparation (Excel)
Data cleaning and preparation steps included:
* Standardizing city names and date formats
* Creating month and year columns** for time-based analysis
* Identifying high-usage customers using IF logic
* Creating pivot tables for city-wise network issues

# SQL Data Analysis
Data was stored and analyzed using SQL:
* Created relational tables with primary and foreign keys
* Wrote joins across customers, usage, billing, and issue tables
* Calculated complaint rate per customer
* Used window functions to analyze monthly complaint trends

# Python (Pandas) Analysis
Further analysis was performed using Python:
* Created customer risk score using custom functions and if–else logic
* Used lambda expressions to identify heavy usage customers
* Performed exploratory data analysis and visualizations

# Tableau Dashboard:
An interactive dashboard was built with:
* Executive KPI Dashboard
* City-wise network performance analysis
* Customer risk and churn indicators

# Business Questions Answered
* Which cities require **urgent network improvement?
* Does poor network quality affect customer payments and churn?
* Are 5G users experiencing fewer network issues compared to other users?

# Outcome
This project helped identify high-risk customers, network problem areas, and usage patterns.
The insights can help telecom companies improve **network quality, customer satisfaction, and retention strategies**.
